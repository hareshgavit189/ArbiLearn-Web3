How It Works

Understanding Proof of Work

The Nonce

Miners increment a number (nonce) millions of times until the SHA-256 hash of the block starts with "00". This is brute-force computation — there's no shortcut.

Chain Linking

Each block stores the previous hash. If you tamper with Block 1, its hash changes — which invalidates Block 2 since Block 2's prevHash no longer matches.

Immutability

To alter Block 1 on a real blockchain, you'd need to re-mine every subsequent block faster than the entire network mines new ones. Mathematically impossible.
