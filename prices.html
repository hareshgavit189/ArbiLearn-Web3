Prices as a Web3 Developer

Gas Cost Awareness

ETH price directly affects gas costs on Ethereum L1. On Arbitrum, gas is paid in ETH too — but at a fraction of the cost. Track ETH to estimate deployment costs.

ARB Token Governance

ARB is the native token of Arbitrum — used for governance votes. Holders decide protocol upgrades, grants, and the future of the network.

CoinGecko API

This dashboard uses CoinGecko's free public API — a rite of passage for every Web3 developer. Fetching on-chain data with no API key is your first step toward building real dApps.
